# Changelog

## 1.0.0 (2026-03-15)

### Features
- **Generate Card** — AI-powered card generation from natural language (Cmd+Shift+A)
- **Embedded Designer** — Split-view card designer with live preview and JSON editor
- **Preview Panel** — Rendered card preview with host config switcher (Teams/Outlook/Webchat)
- **Form Factor Preview** — Desktop, Tablet, Mobile, and Narrow width modes
- **Validate** — Schema validation, accessibility scoring, host compatibility checks
- **Optimize** — Auto-fix accessibility, modernize actions, upgrade version
- **Data to Card** — Convert selected JSON/CSV to optimal card presentation
- **CodeLens** — Preview, Validate, Optimize buttons on .card.json files
- **11 Snippets** — Quick-insert common Adaptive Card elements (ac-card, ac-textblock, etc.)
- **AI Providers** — Optional Copilot, Claude, or OpenAI integration
- **Element Palette** — Quick-insert TextBlock, Image, Table, Input, etc. in Designer
